## 2. 完成内容

### Android 客户端

1. 新增“使用自己的 API Key / 官方 AI 服务”双模式。
2. 自带 Key 模式继续无账号使用，不额外收费。
3. API Key 迁移到 `flutter_secure_storage` / Android Keystore；只有安全写入并回读成功后才清理旧明文。
4. 迁移失败时保留旧配置，避免 Key 丢失；官方账号退出不会删除个人 Key。
5. 新增官方 AI 测试中心：测试账号、服务端报价、模拟订单、影子计费记录、数据删除和账号注销。
6. 报价界面按“分”展示服务端返回的价格明细；本地图表、系统 TTS、本地 OCR 标为免费。
7. 微信支付、支付宝和真实扣费按钮在未配置状态下禁用。
8. 新增隐私、官方服务、退款、数据删除等草案页面，并明确尚未完成法律审核。
9. Debug 构建允许访问模拟器本地测试后端；Release 构建未放开明文 HTTP。
10. 修复英文深色模式下“本周练习趋势”柱形区域纵向溢出。

### FastAPI 后端

1. 新增服务端功能开关，默认安全值：
   - `OFFICIAL_AI_ENABLED=false`
   - `SHADOW_BILLING_ENABLED=true`
   - `PAYMENT_MOCK_ENABLED=true`
   - `WECHAT_PAY_ENABLED=false`
   - `ALIPAY_PAY_ENABLED=false`
   - `REAL_CHARGE_ENABLED=false`
2. 新增报价、订单、生成任务、使用量和审计日志模型。
3. 新增整数分定价：5 题 50 分，超出 5 题后每题 10 分。
4. 新增订单状态机、用户归属检查、报价过期校验、金额复核和幂等控制。
5. 新增 `FakeAiProvider`，支持成功、超时、无效 JSON 和服务商错误，不消耗真实模型额度。
6. 新增 `MockPaymentProvider`，支持支付成功、失败后退款、重复通知和幂等测试。
7. 新增微信、支付宝适配边界；因没有正式商户配置与 SDK 实现，所有真实入口安全返回不可用。
8. 新增影子计费：记录模型、Token、耗时、重试、题数、估算成本和理论毛利，不保存完整资料或完整 Prompt。
9. 新增官方服务数据删除与账号注销接口；账号注销要求再次验证密码。
10. 新增幂等的 Phase 3 SQLite 开发表结构迁移检查。
