## 本轮修复

1. 修复深色模式启动加载页仍使用浅色背景的问题。
2. 修复 API 配置页在深色模式下出现浅色背景、低对比度服务商文字、深浅色卡片混用的问题。
3. API 配置页服务模式、服务商卡片、输入框、教程卡片和说明卡片统一使用主题色。
4. 将外观、界面语言、题目生成语言、OCR 语言、声音、背景音、震动和减少动态效果迁移到独立“偏好设置”页面。
5. 在“我的”页面标题区增加设置入口。
6. 从“我的”学习数据流中移除原内嵌设置卡片，使累计做题统计和练习历史整体上移。
7. 将问题反馈入口移动到练习历史之后，进一步提高学习统计的展示优先级。
8. 修复英文界面下 API 配置页顶部标题仍显示中文的问题。
