> 状态：官网验收候选版

## 本轮定位

v2.0.0 第一阶段从“功能型 AI 出题工具”升级为“个人 AI 学习训练台”的产品结构候选版。

核心路径保持不变：

```text
上传资料 → AI 出题 → 答题练习 → 错题沉淀 → 错题抽卡 → XP 激励 → 持续复习
```

本轮重点不是新增大量功能，而是统一主流程、视觉语言和信息架构。

## Android v2.0.0 已完成内容

### 1. 底部导航收敛为 4 个主入口

```text
首页
出题
错题
我的
```

原来的 API 配置没有删除，而是移动到“我的”页作为设置入口。

### 2. 首页改为学习工作台

首页现在展示：

- 欢迎语和学习状态；
- 总 XP、等级、连续打卡、API 状态；
- 今日 XP、累计做题、正确率、错题数；
- 三倍经验状态和倒计时；
- 上传资料、粘贴资料、开始出题、错题抽卡四个快捷入口；
- 错题抽卡挑战卡片；
- 最近资料；
- 最近练习。

### 3. 出题页改为流程式结构

出题页按步骤组织：

```text
01 选择资料
02 选择题型
03 设置练习参数
04 生成题目
```

保留能力：

- 导入文件；
- 粘贴资料；
- 示例资料；
- 资料下拉选择；
- 删除当前资料（二次确认）；
- 单选、多选、判断、填空、主观题多选；
- 题量加减；
- 5 / 10 / 15 / 20 快捷题量；
- 目标群体选择；
- 生成题目。

### 4. 错题页继续突出错题抽卡

保留 v1.0.6 候选版中的：

- 错题总览；
- 错题抽卡入口；
- 按资料自动分组错题；
- 清空错题二次确认。

### 5. “我的”页承载统计和配置

“我的”页包含：

- XP 等级卡；
- 打卡入口；
- API 配置入口；
- 近 7 天练习趋势；
- 累计做题、正确题数、正确率、错题数；
- 练习历史；
- 关于项目。

## 保持不变的规则

v2.0.0 只优化展示和结构，不修改 XP 规则：

- 每日打卡经验规则沿用 v1.0.5；
- 做题经验规则沿用 v1.0.5；
- 三倍经验规则沿用 v1.0.5；
- 错题抽卡满 5 题且全对，开启 10 分钟三倍经验；
- 三倍经验期间整套练习最终 XP ×3。

## 暂未完成 / 后续事项

### 桌面端

本轮主要完成 Android v2.0.0 第一阶段。桌面端当前仍保留既有 v2-desktop 结构，后续可以按同一设计语言继续升级为：

```text
首页工作台
资料管理
AI 出题
错题抽卡
练习历史
学习统计
API 配置
```

### 抽卡动画

本轮仍没有做抽卡动画专项升级。抽卡视觉入口更突出，但完整的：

- 入场动画；
- 洗牌动画；
- 中央抽取；
- 翻卡发光；
- 抽取成功奖励反馈；

建议后续单独作为动画专项开发。

## 验收重点

请优先检查：

1. 打开 App 后是否明显是“学习工作台”；
2. 底部 4 个入口是否清晰；
3. 首页是否能看到 XP、打卡、三倍经验、错题抽卡；
4. 出题页流程是否比旧版更顺；
5. 资料导入、粘贴、示例是否仍可用；
6. 多题型选择是否仍可用；
7. 删除当前资料是否有二次确认；
8. 错题页按资料分组是否清晰；
9. “我的”页是否能找到 API 配置；
10. XP、打卡、三倍经验逻辑是否未被破坏。
