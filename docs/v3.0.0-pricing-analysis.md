# AI题库 v3.0.0 影子计费与价格分析

## 1. 当前结论

当前价格仅用于开发验证，不是正式商业承诺。所有真实扣费保持关闭，成本数据来自 Fake AI 或服务端估算字段，尚无真实模型样本。

## 2. 初始报价规则

- 5 道普通题：50 分（¥0.50）
- 超出 5 道后：每题增加 10 分（¥0.10）
- 本地 OCR：免费
- 本地图表：免费
- 系统 TTS：免费
- 已有题目练习、错题巩固、闯关和组卷：免费

金额只使用整数分，不使用浮点数。客户端只展示报价，最终金额由服务端创建订单时重新计算。

## 3. 报价样例

| 题量 | 测试报价 |
| --- | ---: |
| 5 题 | ¥0.50 |
| 10 题 | ¥1.00 |
| 20 题 | ¥2.00 |

整套试卷、云端高质量语音、AI 图片、增强 OCR、超长解析和动态关卡暂时只做影子计费，不配置真实售价。

## 4. 影子计费字段

每次官方 AI 任务记录：

- 用户、请求、订单与任务标识
- 服务商与模型
- 输入/输出 Token
- 请求耗时和重试次数
- 请求题数和成功题数
- 估算服务商成本（分）
- 订单报价（分）
- 理论毛利（分）
- 脱敏失败原因与创建时间

默认不记录完整资料、完整 Prompt、API Key 或支付密钥。

## 5. 正式成本计算方法

正式模型接入后，每次任务至少计算：

`模型成本 = 输入Token成本 + 输出Token成本 + 重试成本 + 其他计量能力成本`

`订单总成本 = 模型成本 + 支付通道成本 + 服务器与存储摊销 + 退款风险准备`

`理论毛利 = 实付金额 - 订单总成本`

需分别统计 5、10、20、30 题，以及成功、一次重试、最终失败三类样本。不得仅用单次成功请求推导正式价格。

## 6. 开启真实收费前的数据门槛

1. 至少取得多个真实模型、不同资料长度的影子计费样本。
2. 验证 5 题 ¥0.50 能覆盖长资料、重试与退款冗余。
3. 明确支付通道费率、税务、服务器和客服成本。
4. 统计失败率、无效 JSON 率、重试率和平均生成题数。
5. 价格配置可由服务端版本化、回滚和审计。
6. 报价过期后必须重新获取，订单必须复核报价版本。

在完成这些条件前，`REAL_CHARGE_ENABLED` 必须保持 `false`。
