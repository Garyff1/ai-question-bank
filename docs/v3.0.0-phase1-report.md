# AI题库 v3.0.0 第一阶段开发报告

>
> 基线版本：Android `2.9.9+50`
>
> 开发分支：`codex/v3-phase1-foundation`
>
> 本阶段未发布 GitHub Release，未切换官网 APK。

## 1. 本阶段完成内容

### 架构基础

- 新增 `app/` 应用层，负责应用根节点、设置状态和统一页面过渡。
- 新增 `core/theme/`，集中管理颜色、字号、间距、圆角和动效时长。
- 新增 `core/localization/` 与 ARB 国际化资源。
- 新增 `core/storage/`，为后续数据和安全存储迁移提供统一入口。
- 新增 `core/widgets/`，沉淀卡片、按钮、空状态和加载状态。
- 新增 `features/settings/` 与 `features/challenge/`，新功能不再继续堆入 `main.dart`。
- 保持现有模型字段、SharedPreferences 键和页面入口兼容，没有进行破坏性数据重写。

### 统一设计系统与主题

- 建立浅色、深色、跟随系统三种主题模式。
- 状态栏、系统导航栏、底部导航、卡片、表单、弹窗和主要统计组件同步适配主题。
- 主题偏好保存到本地，重启后保持。
- 将主要页面标题、状态卡、信息卡逐步迁移到 `ColorScheme`，减少写死颜色。
- 增加“减少动态效果”，可关闭页面过渡、卡片入场和闯关节点循环动画。

### 中英文与生成语言

- 接入 Flutter 官方 `gen_l10n`、ARB、`flutter_localizations` 和 `intl`。
- 支持界面语言：跟随系统、简体中文、English。
- 主导航、首页工作台、设置中心、主要页面标题与核心状态文案支持中英文。
- 新增独立“题目生成语言”：跟随资料、中文、英文，并实际注入 AI 出题提示词。
- 用户资料内容与既有 AI 题目不会因界面语言切换而自动翻译。

### 首页工作台

- 新首页包含欢迎区、等级/XP、连续打卡、API 状态。
- 新增今日任务：闯关、错题复习、每日打卡和生成练习。
- 新增继续挑战卡，显示当前章节、关卡类型和总进度。
- 快捷入口扩展为上传、粘贴、扫描占位、出题、错题抽卡和闯关。
- OCR 扫描入口按计划保留 Phase 2 提示，本阶段不提前接入实现。

### 闯关模式升级

- 每章固定五种关卡规则：热身、连击、错题伏击、生存、Boss。
- 热身关 3 题，其余关卡 5 题；闯关题型继续明确排除听力题。
- 增加三格知识护盾、连击/最高连击、Boss 能量、限时生存和薄弱知识点记录。
- 护盾归零使用温和失败提示，并提供重新挑战、错题巩固和返回地图。
- 结果页增加正确率、最高连击、剩余护盾、薄弱知识点、星级、XP 与下一关操作。
- 保留现有章节地图、星级、徽章、XP 和 `rpg_progress_v1` 数据。
- 修复结果弹层按钮回调误关闭宿主路由的回归风险，并增加按钮级组件测试。

### 设置中心

- 新增外观、界面语言、题目语言、音效、背景音、震动和减少动态效果设置。
- AI 服务区明确区分“自己的 API Key”和“官方 AI 服务（即将开放）”。
- 增加 Flutter 开源许可页面入口。

## 10. 当前遗留问题

1. `main.dart` 已建立模块化迁移边界并拆出 v3 新模块，但遗留页面仍然很大。为了避免一次性迁移破坏旧数据和复杂 PDF/TTS 流程，后续仍需按 feature 逐步搬迁，不能将本阶段误解为完成了全部历史代码拆分。
2. 主导航、首页、设置和主要页面标题已国际化；部分旧版深层卡片、历史统计细项、旧弹窗仍保留中文。正式 `3.0.0` 前需要继续清点 ARB 覆盖率。
3. 部分旧组件仍有写死颜色，关键主流程已适配深色主题，深层旧页面仍需逐页视觉回归。
4. `flutter analyze` 的 29 条 info 主要来自旧代码异步 context 和风格规则，不影响当前构建，但应随模块拆分逐步清理。
5. API Key 安全存储、OCR、富媒体重构和开源许可清单属于后续阶段，不在 Phase 1 提前实现。

## 11. 第二阶段建议

在用户确认 Phase 1 的 APK 体验后再进入 Phase 2，优先顺序建议：

1. 完成剩余主流程国际化和深色视觉清单。
2. 接入本地 OCR，并先保证识别结果可编辑、可保存、可用于出题。
3. 统一图表结构并验证答题页、解析页、PDF 三处一致。
4. 评估从 `flutter_edge_tts` 迁移到稳定 TTS 方案，同时保留旧听力题兼容。
5. 将错题知识点柱状图改为可操作诊断列表。
6. 完成 `THIRD_PARTY_NOTICES.md` 与 Syncfusion 许可风险核查。
