## Syncfusion PDF 处理方案

必须选择并留档其中一个方案：

1. 申请并保存有效的 Syncfusion Community License 批准记录；
2. 购买商业许可证；
3. 替换 PDF 引擎并重新验证全部试卷能力。

替换 PDF 引擎会影响：题目和答案分页、中文字体、数学公式、图表、图片、音频提示、旧试卷兼容和打印效果。未经完整回归，不得直接替换后上线。
