API Key 获取与配置指南

如果你是第一次接触 API Key，可以把它理解成“模型平台给你的调用密码”。先去服务商控制台创建 Key，再回到 AI题库的“API 配置”页填入 Key、Base URL 和模型名称，最后点“测试连接”。

① DeepSeek

登录 DeepSeek Platform，进入 API Keys 页面创建 Key；Base URL 填 https://api.deepseek.com。

打开 DeepSeek API Keys ↗
② Qwen / 阿里百炼

登录阿里云百炼控制台，选择地域后进入 API Key 页面，点击“创建 API Key”。

查看百炼获取 Key 文档 ↗
③ 智谱 / Z.ai

登录 Z.ai Open Platform，在 API Keys 管理页创建 Key；Base URL 填 https://api.z.ai/api/paas/v4。

查看 Z.ai Quick Start ↗
④ 小米 MiMo

进入 MiMo API Keys 创建 Key。普通按量付费使用 https://api.xiaomimimo.com/v1，模型可先填 mimo-v2.5-pro。

查看 MiMo 首次调用文档 ↗
⑤ Kimi / Moonshot

登录 Kimi 开放平台控制台创建 Key；Base URL 填 https://api.moonshot.ai/v1。

查看 Kimi API Overview ↗
⑥ 自定义接口

如果你的服务商兼容 OpenAI Chat Completions，通常只需要填自己的 Key、Base URL 和模型名。

不确定怎么填？提交 Issue ↗
Key 只保存在你的设备本地，不会上传到 AI题库服务器。 创建 Key 后请立刻复制保存，很多平台关闭弹窗后无法再次查看明文 Key。 如果测试失败，优先检查余额/额度、Base URL、模型名和 Key 是否复制完整。
