# AI题库 · 项目概况

> 用于发给网页设计师，以此设计完整的网站页面布局和交互。

---

## 一、产品定位

一款面向全年龄段学生的**AI智能出题练习平台**。用户上传学习资料（PDF/Word/TXT），AI 基于资料内容自动生成选择题和填空题，用户逐题作答并查看解析。核心价值：**把自己的学习资料变成智能题库**。

目标用户：小学生、初中生、高中生、大学生、考研考证人群。

---

## 二、技术架构

| 层 | 技术 | 说明 |
|----|------|------|
| 后端 | Python FastAPI | 文件解析、AI出题、答题记录、数据存储 |
| 数据库 | SQLite | 本地存储，无需云端 |
| AI | DeepSeek API | 通过 OpenAI 兼容接口调用 |
| 前端 | 纯 HTML/CSS/JS | 单页应用，移动端优先响应式 |

## 三、功能模块 & 页面

### 共 6 个核心页面 + 1 个弹窗系统

### 1. 登录/注册页
- 两侧有柔和的渐变色块（蓝色 + 琥珀色光斑）作为背景装饰
- 品牌 Logo 大字「AI题库」+ 副标题「智能出题 · 高效学习」
- 登录/注册两个 Tab 切换
- 账号和密码两个输入框，随便填都能用（云端版再完善）
- 一个大的提交按钮

### 2. 主页 - 资料列表
- 顶栏：标题「我的资料」+ 右上角「＋」上传按钮
- 上传规格提示：支持 PDF、Word、TXT，单个文件不超过 20MB
- 资料卡片列表（竖排），每个卡片包含：
  - 左侧：圆形色块文件图标（PDF蓝、Word绿、TXT灰）
  - 中部：文件名 + 上传时间
  - 右下角：两个小按钮「出题」「✕删除」
- 空状态：一个大图标 + 引导文案
- 底部固定导航栏：主页 / 历史 / 错题本 / 统计

### 3. 生成题目页 ⭐
- 顶部有返回按钮
- 当前选中的资料名芯片标签
- **题型选择**：选择题（四选一）/ 填空题（手动输入），卡片式切换，选中态蓝色边框
- **目标群体**：下拉选择框，可选 小学1-2年级/3-4年级/5-6年级/初中/高中/大学生/通用
- **题目数量**：步进器（- 5 +），范围 1-20
- 底部「生成题目」大按钮，点击后显示加载状态，AI 出题完成后自动跳转答题页

### 4. 答题练习页 ⭐⭐ 核心体验
- 顶栏：返回按钮 + 进度文字「2/5」+ 连胜计数器 🔥
- 进度点：一排小圆点，做完的变绿色（正确）或红色（错误），当前题目蓝色
- **题目卡片**：浅米色背景，题号 + 题目正文，带阴影
- **选项列表**：选择题显示可点击选项，填空题显示输入框
- **提交反馈动画**⭐：
  - 提交后印章从上方砸下（✅绿色/❌红色），带旋转和细微墨点粒子效果
  - 答错时选项变红，正确选项变绿
  - 答错时题目卡片轻微抖动
  - 显示「正确答案」和「💡解析」
- 完成页面：🎉 纸屑庆祝动画 + 环形分数图 + 数字滚动递增 + 「再来一轮」「返回主页」

### 5. 练习历史
- 所有练习按时间倒序排列
- 每条记录显示：「教材名 #序号」+ 正确率百分比 + 正确数/总题数 + 错题数 + 时间
- 同一教材多次练习会递增序号（如 test_data #1, test_data #2）

### 6. 错题本 ⭐
- 按教材分组显示（每个教材一个卡片组）
- 每组显示教材名 + 错题数标记 + **「重练」按钮**
- 每条错题显示：题号、题目原文、选项、正确答案、💡知识点解析
- 点击「重练」按钮 → 提取该教材所有错题 → 进入答题模式专项训练

### 7. 学习统计
- 环形图显示总正确率（带数字滚动动画）
- 四行数据：累计做题、正确题数、练习次数、最近练习时间

### 8. 弹窗系统
- 退出确认弹窗（答题页返回时触发）
- 删除确认弹窗（删除资料时触发，红色危险按钮）
- 退出登录确认

## 五、设计风格关键词

**简洁 · 趣味 · 教育感 · 卡片式 · 移动优先**

- 圆角规范：卡片 14-16px、按钮 12px、标签 8-10px、圆形图标
- 阴影规范：卡片使用 `box-shadow: 0 6-16px rgba(15,23,42,.18)` 柔阴影
- 按钮阴影：主色按钮附带蓝色阴影 `box-shadow: 0 10px 20px -8px rgba(37,99,235,.5)`
- 动效风格：微缩放点击反馈、印章砸下弹跳、纸屑飘落庆祝、数字滚动递增
- 响应式：以 480px 宽度为设计基准（手机），同时兼容桌面端展示

## 七、特殊交互说明（设计时注意）

### 答题反馈（核心体验）
- **印章动画**：答对/答错瞬间，一枚带毛边质感的印章从题卡上方"咚"地砸下来，带细微墨点飞溅
- 答错时题卡轻微抖动一下（不是惩罚感的剧烈晃动，是带点幽默的"哎呀"感）
- 所有中国学生最熟悉的"老师阅卷盖章"场景，比通用勾叉更有记忆点

### 连胜计数器 🔥
- 连续答对时火苗图标旁的数字递增
- 根据连胜值轻微放大，答错时"啪"一下熄灭归零

### 纸屑庆祝 🎉
- 完成一轮练习后彩色小纸屑从屏幕顶部落下
- 环形分数图从0%动画绘制到最终值
- 百分比数字逐位递增
