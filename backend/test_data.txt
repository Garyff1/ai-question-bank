Python 是一种高级编程语言，由 Guido van Rossum 于 1991 年首次发布。Python 的设计哲学强调代码的可读性和简洁性，语法使用缩进来表示代码块。

Python 支持多种编程范式，包括面向对象、命令式、函数式和过程式编程。它拥有一个庞大而标准库，被称为"batteries included"（自带电池）。

Python 的常见数据类型包括：int（整数）、float（浮点数）、str（字符串）、list（列表）、dict（字典）、tuple（元组）和 set（集合）。

Python 中的列表是可变的序列，使用方括号 [] 创建。字典使用键值对存储数据，用花括号 {} 创建。

函数使用 def 关键字定义，类使用 class 关键字定义。Python 使用 try/except 进行异常处理。

Python 广泛应用于 Web 开发、数据科学、人工智能、自动化脚本等领域。
